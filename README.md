# Sales Performance Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes sales performance, revenue trends, customer behavior, and product performance using retail sales data. The dashboard provides interactive insights into revenue generation, profitability, customer contributions, and regional performance.

## Dashboard Screenshot

![Dashboard](Screenshots/sales_dashboard.png)

## Tools Used

* Power BI
* Excel
* SQL

## Key Metrics

* Total Revenue
* Total Profit
* Total Orders
* Sales Quantity

## Key Insights

* West region generated the highest revenue contribution.
* Technology category contributed the largest share of sales.
* Top customers significantly influenced overall revenue.
* Monthly sales trends indicate seasonal fluctuations and growth opportunities.
* Product-level analysis helps identify high-performing products and revenue drivers.

## Business Impact

This dashboard enables data-driven decision-making by helping businesses monitor sales performance, identify growth opportunities, optimize product strategies, and improve customer-focused planning.
